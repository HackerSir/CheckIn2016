## Framework
- Laravel 5.2
- Semantic UI 2.2

## Requirement
For Server: 
- Apache >= 2.4 or Nginx >= 1.8
- PHP >= 5.6
- MariaDB >= 10.0 or MySQL >= 5.7
- NodeJS >= 5.6
  
For Client:
- IE >= 9
- Chrome / Firefox is fine.
